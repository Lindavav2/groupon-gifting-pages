The Groupon Gift Card

A digital prepaid gift card that unlocks Groupon's full catalogue for the recipient — the ideal gift for any occasion, delivered instantly.

Key Features

  • Fully digital — instant code delivery, no physical fulfilment required
  • Full card value credited as Groupon Bucks
  • Redeemable across the entire Groupon catalogue (excluding purchase of another gift card and a very small number of select partner offers)
  • Gift cards are valid for a set period — see market-specific terms

How Redemption Works

1
Recipient visits groupon.com/giftcards
2
Recipient enters the unique gift card code
3
Full value is credited to their account as Groupon Bucks
4
Bucks are applied at checkout on any eligible deal
Groupon Bucks can be combined with other promotions — maximising the value for every recipient.
